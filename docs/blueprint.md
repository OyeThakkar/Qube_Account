# **App Name**: Qube Central

## Core Features:

- Centralized Dashboard: Display key metrics on a centralized dashboard: total companies, total users, active services.
- Company Directory: Provide a company listing page to enable users to Add a New Company, Edit Company details, or Deactivate an existing company.
- Portal User Management: Provide a central portal for internal (Qube Cinema) users who manage the Qube Account portal, so these users can be managed independently of individual company users.
- Services Listing: Display a read-only table of services, with details such as Service Name, Access URL, # of Subscribed Companies (active only), and Last Updated date
- Settings: Centralize all internal (Qube Cinema) admin functions, such as Authentication settings, notification preferences, and an audit log

## Style Guidelines:

- Use a primary blue color (#007bff) to establish trust and professionalism.
- Implement a secondary gray color (#6c757d) for a clean and modern aesthetic.
- Accent with green (#28a745) to indicate success and positive actions.
- Employ clear, sans-serif fonts for readability.
- Utilize minimalist icons for intuitive navigation.
- Use a consistent and clean grid layout for each section, for a professional and scalable interface.
